# Add ibl.ai Profile

Add profile features to your ibl.ai app — a full profile settings page
with tabs for Basic info, Social links, Education, Experience, Resume,
and Security, plus a compact avatar dropdown for your navbar.

## Profile Settings Page

A full-featured profile settings page using the SDK's `Profile` component.

### Prerequisites

The following packages are already included in apps generated by
`iblai startapp agent` — no extra `pnpm add` needed:

- `@tanstack/react-form`, `zod`, `react-gravatar`
- `@radix-ui/react-checkbox`, `@radix-ui/react-dialog`, `@radix-ui/react-select`, `@radix-ui/react-tabs`

### Create `app/(app)/profile/page.tsx`

```tsx
"use client";

import { useEffect, useState } from "react";
import { useRouter } from "next/navigation";
import { Profile } from "@iblai/iblai-js/web-containers";
import { config } from "@/lib/config";

function resolveTenantKey(raw: string | null): string {
  if (!raw || raw === "[object Object]") return "";
  try {
    const p = JSON.parse(raw);
    if (typeof p === "string") return p;
    if (p?.key) return p.key;
  } catch {}
  return raw;
}

export default function ProfilePage() {
  const router = useRouter();
  const [username, setUsername] = useState("");
  const [tenantKey, setTenantKey] = useState("");
  const [isAdmin, setIsAdmin] = useState(false);
  const [ready, setReady] = useState(false);

  useEffect(() => {
    try {
      const raw = localStorage.getItem("userData");
      if (raw) {
        const parsed = JSON.parse(raw);
        setUsername(parsed.user_nicename ?? parsed.username ?? "");
      }
    } catch {}

    const stored =
      localStorage.getItem("current_tenant") ??
      localStorage.getItem("tenant");
    const resolved = resolveTenantKey(stored) || config.mainTenantKey();
    setTenantKey(resolved);

    try {
      const tenantsRaw = localStorage.getItem("tenants");
      if (tenantsRaw) {
        const tenants = JSON.parse(tenantsRaw);
        const match = tenants.find((t: any) => t.key === resolved);
        if (match) setIsAdmin(!!match.is_admin);
      }
    } catch {}

    setReady(true);
  }, []);

  if (!ready || !username || !tenantKey) {
    return (
      <div style={{ height: "100vh", width: "100vw", display: "flex", alignItems: "center", justifyContent: "center" }}>
        <p style={{ color: "#9ca3af", fontSize: "0.875rem" }}>Loading profile…</p>
      </div>
    );
  }

  return (
    <div style={{ height: "100vh", width: "100vw", overflow: "auto" }}>
      <Profile
        tenant={tenantKey}
        username={username}
        isAdmin={isAdmin}
        onClose={() => router.push("/")}
        customization={{ showUsernameField: false, showPlatformName: true, useGravatarPicFallback: true }}
        targetTab="basic"
      />
    </div>
  );
}
```

### `<Profile>` Props

| Prop | Type | Description |
|------|------|-------------|
| `tenant` | `string` | Tenant/org key |
| `username` | `string` | Username |
| `isAdmin` | `boolean` | Shows admin badge + settings |
| `onClose` | `() => void` | Cancel callback |
| `targetTab` | `string` | Initial tab: `basic`, `social`, `education`, `experience`, `resume`, `security` |

## Profile Dropdown (Navbar Component)

A compact avatar dropdown with profile link, tenant switching, and logout.

### Steps

1. Call `get_component_info(componentName: "UserProfileDropdown")` for the full props reference
2. Import SDK styles: `@import "@iblai/iblai-js/web-containers/styles";`
3. Create `components/iblai/profile-dropdown.tsx` using `UserProfileDropdown` from `@iblai/iblai-js/web-containers/next`
4. Place in your navbar:

```tsx
import { ProfileDropdown } from "@/components/iblai/profile-dropdown";

<ProfileDropdown />
```

## Important Notes

- **Redux store**: Must include `mentorReducer` and `mentorMiddleware`
- **`initializeDataLayer()`**: 5 args (v1.2+)
- **`@reduxjs/toolkit`**: Deduplicated via webpack aliases
- **Admin detection**: Derive from `tenants` array in localStorage
